--------------------------------------------------------
        Python OOP Project : Employee Management System
--------------------------------------------------------

Project Description:
--------------------
This project is developed using Object Oriented Programming (OOP) concepts
in Python. It demonstrates inheritance, encapsulation, constructors,
destructors, method overriding, and getter/setter methods.

Classes Used:
-------------

1. Employee (Base Class)
   Attributes:
   - employee_id (Private)
   - name
   - age
   - salary (Private)

   Methods:
   - get_employee_id()
   - set_employee_id()
   - get_salary()
   - set_salary()
   - display()
   - show_details()
   - __del__() (Destructor)

2. Developer (Derived Class)
   Inherits from Employee.

   Additional Attribute:
   - language

   Additional Methods:
   - display() (Method Overriding)
   - show_details()

3. Manager (Derived Class)
   Inherits from Employee.

   Additional Attribute:
   - department

   Additional Methods:
   - display() (Method Overriding)
   - show_details()

Features:
---------
1. Create a Person.
2. Create a Developer.
3. Create a Manager.
4. Show details of Person, Developer, or Manager.
5. Exit the program.

OOP Concepts Implemented:
-------------------------
✔ Class and Object
✔ Constructor
✔ Destructor
✔ Inheritance
✔ Encapsulation (Private Attributes)
✔ Getter and Setter Methods
✔ Method Overriding
✔ Use of self keyword

How to Run:

1. Open the Python file.

3. Select an option from the menu.
4. Enter the required details.
5. View the created object details using option 4.
